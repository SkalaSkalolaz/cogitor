# cogitor
AI Cogitor — интеллектуальный ассистент для разработчиков с интеграцией LLM (Large Language Models). Поддерживает C LI, веб-интерфейс и десктопный режим. Генерирует, анализирует и исправляет код на множестве языков программирования  с автоматическим поиском в интернете и RAG-режимом для работы с локальными данными.
